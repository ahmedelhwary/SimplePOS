# Simple ERP - Sales Invoice Module

## Technologies Used
- .NET 10 Windows Forms
- Entity Framework Core (Code First)
- SQL Server

## How to Run

1. Open solution in Visual Studio
2. Update connection string in ERPDbContext if needed
3. Run migration or execute attached DatabaseScript.sql
4. Run the project

## Features
- Create new invoice
- Auto invoice number generation
- Add multiple items
- Automatic total calculation
- Save invoice with details
- Delete invoice
